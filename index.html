TSM Functions

first(n...)

Return first valid n.

min(n...)

Return lowest n.

max(n...)

Return highest n.

avg(n...)

Return average of all valid n.

ifeq(a, b, x, y)

If a is equal to b return x, else return y or invalid if y is not defined.

ifgt(a, b, x, y)

If a greater than b return x, else return y or invalid if y is not defined.

ifgte(a, b, x, y)

If a greater than or equal to b return x, else return y or invalid if y is not defined.

iflt(a, b, x, y)

If a is less than b return x, else return y or invalid if y is not defined.

iflte(a, b, x, y)

If a is less than or equal to b return x, else return y or invalid if y is not defined.

round(x, f)

Round x to nearest multiple of f or to nearest copper if f is not defined.

roundup(x, f)

Round x up to nearest multiple of f or to nearest copper if f is not defined.

rounddown(x, f)

Round x down to nearest multiple of f or to nearest copper if f is not defined.

convert(x)

Return lowest cost of x by conversion via milling or prospecting or transforming (splitting, combining) or ink vendor prices.

check(a, b, c)

If a is invalid, return invalid. If a is greater than 0 return b. Else return c or invalid if c is not defined.
